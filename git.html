Comando Función
git --version Muestra información sobre la versión de Git.
git config --help Se puede obtener una lista de todas las variables de configuración disponibles mediante el comando git help.
git config --global user.name "Juanito" Puedes hacer que Git lea y escriba específicamente en este archivo pasando la opción --global, cómo el nombre de usuario de Github.
git config --global use.email "Juanito.el.más.capo@gmail.com" Puedes hacer que Git lea y escriba específicamente en este archivo pasando la opción --global, cómo el correo electrónico.
git config -l Permite obtener y establecer variables de configuración que controlan el aspecto y funcionamiento de Git.
cd Te permite navegar entre carpetas del ordenador.
ls Muestra información sobre los archivos en el índice y el árbol de trabajo.
mkdir Crea una carpeta en la ubicación actual.
start . Abre la carpeta en la cual se ubica la consola.
git init inicia el repositorio.
cd .. Permite retroceder la ubicación en carpetas.
git add . Añade los archivos actualizados al repositorio.
git commit -m Permite hacer un comentario en el repositorio.
git status Permite ver el estado de los archivos en el repositorio, si han o no han sido actualizados.
git log Genera un historial de uso u otras informaciones del repositorio.
